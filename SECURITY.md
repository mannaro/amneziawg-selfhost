# Security Policy

## Никогда не публикуйте

- PrivateKey
- PresharedKey
- реальные client.conf
- backup-файлы
- params с рабочего сервера
- пароли
- токены
- реальные IP-адреса без необходимости

## Можно публиковать

- example-конфиги
- шаблоны
- обезличенные примеры
- Mermaid-схемы
- скриншоты без секретов

Перед публикацией всегда проверяйте содержимое репозитория.
