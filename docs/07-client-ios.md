# 07. iPhone Client

> 🚧 **Раздел находится в разработке.**
>
> Эта глава будет опубликована после тестирования клиента AmneziaWG на iPhone под управлением iOS. Содержание может дополняться и изменяться.

---

# 🎯 Что будет рассмотрено

В этой главе будут рассмотрены:

- установка приложения AmneziaWG на iPhone;
- импорт клиентского профиля;
- первое подключение к VPN;
- проверка работы соединения;
- настройка автоматического подключения;
- рекомендации по использованию в мобильных сетях и Wi-Fi;
- обновление клиента;
- диагностика распространённых проблем.
