# 11. Troubleshooting

> 🚧 **Раздел находится в разработке.**
>
> Эта глава будет постоянно дополняться по мере накопления практического опыта эксплуатации AmneziaWG. Здесь будут собраны наиболее распространённые проблемы, способы их диагностики и варианты решения.

---

# 🎯 Что будет рассмотрено

В этой главе будут рассмотрены:

- диагностика проблем с подключением к серверу;
- анализ логов AmneziaWG;
- проверка состояния служб;
- проблемы с DNS и их устранение;
- ошибки маршрутизации и Split Routing;
- проблемы с брандмауэром;
- диагностика MTU и фрагментации пакетов;
- ошибки подключения клиентов Windows и iOS;
- диагностика OpenWrt;
- распространённые ошибки и способы их устранения.

---

# 📋 Статус

- 🚧 Подготовка материала
- ⏳ Сбор типовых ошибок
- ⏳ Проверка решений
- ⏳ Подготовка примеров
- ⏳ Добавление полезных команд
- ⏳ Финальное редактирование

# 🎉 Завершение

Поздравляем!

Если вы дошли до этой главы, значит:

- сервер AmneziaWG успешно развернут;
- веб-панель настроена;
- клиентские устройства подключены;
- маршрутизация работает;
- при необходимости настроен OpenWrt;
- подготовлены процедуры резервного копирования;
