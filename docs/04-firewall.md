# 04. Настройка Firewall

> 🚧 **Раздел находится в разработке.**
>
> Эта глава будет опубликована после проверки конфигурации на рабочем сервере. Содержание может дополняться и изменяться.

---

# 🎯 Что будет рассмотрено

В этой главе будут рассмотрены:

- установка и настройка `nftables`;
- создание базовой политики фильтрации;
- защита SSH-подключения;
- открытие UDP-порта AmneziaWG;
- ограничение доступа к Web Panel;
- разрешение трафика VPN-клиентов;
- сохранение правил;
- автоматическая загрузка Firewall после перезагрузки;
- проверка и диагностика правил.
