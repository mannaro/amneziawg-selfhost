# 08. OpenWrt (Cudy)

> 🚧 **Раздел находится в разработке.**
>
> Эта глава будет опубликована после завершения тестирования роутера **Cudy WR3000S** с **OpenWrt** и **AmneziaWG**. Все инструкции будут основаны на реальной рабочей конфигурации.

---

# 🎯 Что будет рассмотрено

В этой главе будут рассмотрены:

- установка OpenWrt на роутер Cudy WR3000S;
- первоначальная настройка системы;
- установка клиента AmneziaWG;
- импорт клиентской конфигурации;
- настройка постоянного VPN-подключения;
- настройка Policy Based Routing (PBR);
- использование Split Routing;
- проверка подключения домашней сети через VPN;
- диагностика распространённых проблем.
