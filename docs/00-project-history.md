# История проекта

## Почему появился этот проект

Этот репозиторий появился не как учебный пример и не как пересказ официальной документации.

Его цель — документировать полностью рабочую инфраструктуру AmneziaWG, которая используется в реальной эксплуатации.

Во время настройки пришлось решить множество задач, которые редко подробно описываются в одном месте:

- подготовка VPS;
- установка AmneziaWG;
- настройка Web Panel;
- ограничение доступа к панели управления;
- Split Routing;
- подключение Windows;
- подключение iPhone;
- подключение OpenWrt (Cudy);
- интеграция с Podkop;
- резервное копирование;
- восстановление после сбоя.

Каждый раздел этого проекта основан на реально проверенной конфигурации.

Главная цель — чтобы по этой документации можно было полностью восстановить сервер с нуля.
