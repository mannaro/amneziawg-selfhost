# 06. Windows Client

> 🚧 **Раздел находится в разработке.**
>
> Эта глава будет опубликована после тестирования клиента AmneziaWG в Windows 11. Содержание может дополняться и изменяться.

---

# 🎯 Что будет рассмотрено

В этой главе будут рассмотрены:

- установка клиента AmneziaWG для Windows;
- импорт клиентского профиля;
- первое подключение к серверу;
- проверка работы VPN;
- проверка публичного IP-адреса;
- настройка автоподключения;
- обновление клиента;
- диагностика распространённых проблем.
